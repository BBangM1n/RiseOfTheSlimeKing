# RiseOfTheSlimeKing
## 게임 장르 : 2D RPG
## 게임 소개 : 숲속에 숨어있던 슬라임들이 '무언가'에 의해 점점 이상현상을 보이는 스토리의 2D 성장형 방식 RPG 게임입니다.
## 개발 목적 : RPG게임의 특징들을 축소해서 일부 구현 목표를 잡고 개발하였습니다.
## 사용 엔진 : UNITY 2022.3.62f2
## 개발 기간 : 2025.11.22 ~ 2024.01.22
## 사용 기술
- Unity(C#)
- Unity Addressables
- Unity UI / EventSystem
- CSV 기반 데이터 관리
- Object Pooling
- LINQ
- PlayerPrefs
- JSON 기반 세이브 / 로드 형식
